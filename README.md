# Student Scholarship Portal

A web-based scholarship management system developed as a university project.  
The portal allows students to apply for scholarships online while enabling efficient management and storage of scholarship records through a database-driven system.

---

## Features

- Student scholarship application form
- Scholarship record management
- Database integration for storing student information
- Form validation
- User-friendly interface
- Responsive design

---

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL

---

## Project Purpose

The purpose of this project was to create a digital solution for managing scholarship applications efficiently and reducing manual paperwork through an online system.
